Agreed-upon content pipeline as of 2026-04-14:

*Production (la-force channel):*
• Daily cron generates "man bites dog" collision narratives — provocative candidates that make people sit up
• Example of the tone: the truth narratives / epistemic collapse collision

*Editorial gate (Curtis):*
• Curtis reviews daily candidates and selects the strongest ones
• When one is worth sharing, la-force pings ObiWan (this channel) with the collision details

*Distribution (this channel — Mind Tricks / ObiWan):*
• ObiWan reads the collision, pulls out the sharpest provocation
• Drafts LinkedIn package: post copy, 60-90 sec video talking points, first-comment text
• Curtis and ObiWan refine together, Curtis records and posts

*LinkedIn format (planned, not yet deployed):*
• Native video post (60-90 sec, Curtis on camera explaining why the collision matters)
• Post text frames the provocation
• First comment drops the full collision brief as PDF/link
• This two-layer format is algorithm-favored: video = priority reach, comment = engagement + dwell time

*InnoLead channel:*
• Bi-weekly signal stories sent to Scott Kirsner for newsletter placement
• Same collision engine feeds this channel — minimal rework needed

*Why:* One production pipeline feeding multiple distribution channels. The daily cron creates volume, Curtis's judgment selects for quality, ObiWan handles marketing shaping.

*How to apply:* After the first live run through this pipeline, ObiWan will formalize it as a repeatable skill.
