# Hamid's Thinker Portfolio

250 thinkers across 9 domains. This is the full intellectual palette for making cross-domain
connections in the digest. Rotate across it — never default to the same few names.

## How to Use This File

Before writing each issue:
1. Pick 3-5 thinkers you haven't referenced recently. Let them shape this issue's connections.
2. Use **SYNERGIES** to find unexpected pairings between thinkers from different domains.
3. Use **TENSIONS** to frame ironic contradictions between signals.
4. Match **METHOD** to signal type (empirical findings → empirical thinkers, paradigm shifts → critical/synthetic thinkers).
5. Match **SCALE** to signal scope (individual behavior → Individual, org change → Organizational, cultural shifts → Societal).
